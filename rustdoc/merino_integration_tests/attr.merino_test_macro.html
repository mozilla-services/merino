Wrap a test case in setup and tear down boiler plate, including manipulating settings if needed.

If other test macros are used, such as the parameterized crate, the attributes for those macros should be placed below the merino_test macro. Additionally, the settings closure can specify arguments that will be available on the resulting function, for use by parameterized.
