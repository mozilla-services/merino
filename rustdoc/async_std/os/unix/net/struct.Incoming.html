pub struct Incoming<'a> { /* fields omitted */ }
Expand description

A stream of incoming Unix domain socket connections.

This stream is infinite, i.e awaiting the next connection will never result in None. It is created by the incoming method on UnixListener.

This type is an async version of std::os::unix::net::Incoming.
