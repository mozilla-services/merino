pub enum ParseResult<T, E> {
    CommitOk(T),
    PeekOk(T),
    CommitErr(E),
    PeekErr(Tracked<E>),
}
Expand description

A Result type which has the committed status flattened into the result. Conversions to and from std::result::Result can be done using result.into() or From::from(result)

Variants

CommitOk(T)

Tuple Fields

0: T

The parser has succeeded and has committed to this parse. If a parser after this fails, other parser alternatives will not be attempted (CommitErr will be returned)

PeekOk(T)

Tuple Fields

0: T

The parser has succeeded and has not committed to this parse. If a parser after this fails, other parser alternatives will be attempted (PeekErr will be returned)

CommitErr(E)

Tuple Fields

0: E

The parser failed other parse alternatives will not be attempted.

PeekErr(Tracked<E>)

Tuple Fields

0: Tracked<E>

The parser failed but other parse alternatives may be attempted.
