Calculates the edit distance and the changeset for two given strings. The first string is assumed to be the ‚Äúoriginal‚Äù, the second to be an edited version of the first. The third parameter specifies how to split the input strings, leading to a more or less exact comparison.

Common splits are "" for char-level, " " for word-level and "\n" for line-level.

Outputs the edit distance (how much the two strings differ) and a ‚Äúchangeset‚Äù, that is a Vec containing Differences.

Examples

use difference::diff;
use difference::Difference;

let (dist, changeset) = diff("test", "tent", "");

assert_eq!(changeset, vec![
    Difference::Same("te".to_string()),
    Difference::Rem("s".to_string()),
    Difference::Add("n".to_string()),
    Difference::Same("t".to_string())
]);
