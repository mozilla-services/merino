Crate sentry_anyhow[−][src]

Expand description

Adds support for capturing Sentry errors from anyhow::Error.

Example

use sentry_anyhow::capture_anyhow;
let result = match function_that_might_fail() {
    Ok(result) => result,
    Err(err) => {
        capture_anyhow(&err);
        return Err(err);
    }
};
